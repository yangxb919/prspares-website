### D. 移动端表单优化 - 加 autoComplete + 防 iOS 自动 zoom（已实施）

- 涉及路径：`src/app/wholesale-inquiry/page.tsx`
- 改动 1：**字体 14px → 16px (mobile only)**
  - 把所有 input/select/textarea 的 `text-sm` 改成 `text-base sm:text-sm`
  - 原因：iOS Safari 在 input font-size < 16px 时点击会**自动 zoom 整个页面**，造成填表中页面突然跳动放大、用户必须双指反向缩回。这是经典的移动 UX 问题，行业广泛公认 ≥16px 是 iOS input 最佳实践
- 改动 2：**加 `autoComplete` 属性** 5 个 input
  - name → "name"
  - email → "email" + `inputMode="email"`
  - phone → "tel" + `inputMode="tel"`
  - company → "organization"
  - country → "country-name"
  - 作用：让移动浏览器密码管理器 / iOS Keychain / Chrome 自动填充工作，省去用户手动输 5 个字段
- 表单架构已经不错（必填只 4 个、可选字段折叠、grid 响应式），所以不做大改动
- **预期效果**：移动端（占 Ads 流量 82%）填表流畅度大幅提升，begin_form → generate_lead 漏斗成功率再提高 5-10pp
